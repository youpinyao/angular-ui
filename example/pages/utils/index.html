angular 扩展

angular.each

@params {Array, Object} item 要遍历的数据或对象 @params {Function} callback 要遍历的数据或对象的回调 @params {Number, String} value 遍历数组或对象每个的value @params {Number, String} key 遍历数组或对象每个的key const items = [1, 2, 3]; angular.each(items, (value, key) => { console.log(value, key); });

angular.isNull

@params value 要判断的值 @return {Boolean} 是否为空 const value = ''; console.log(angular.isNull(value));

angular.isEmpty

@params value 要判断的值，可判断空json和数组 @return {Boolean} 是否为空 const value = {}; console.log(angular.isEmpty(value));

angular.isArray

@params value 要判断的值 @return {Boolean} 是否为空 const value = ''; console.log(angular.isArray(value));

angular.uuid

@return {String} 随机生成的uuid const value = ''; console.log(angular.uuid());
