Alert 警告提示

当某个页面需要向用户显示警告的信息时。
非浮层的静态展现形式，始终展现，不会自动消失，用户可以点击关闭。

代码演示

${require('./code.html')}
jsfiddle
// ma-items $scope.alertItems = [{ type: 'success', text: '成功提示框，主要用来显示成功信息。', close: false, }, { type: 'danger', text: '危险提示框，主要用来显示错误信息。', }, { type: 'warning', text: '警告提示框，主要用来显示重要信息。', }];

API

属性 说明 类型 默认值
ma-items 提示文本对象 Array -
ma-class 此样式会应用于alert的每个item String -
