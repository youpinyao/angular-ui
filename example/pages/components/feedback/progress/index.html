Progress 进度条

在操作需要较长时间才能完成时，为用户显示该操作的当前进度和状态。
当一个操作会打断当前界面，或者需要在后台运行，且耗时可能超过2秒时；
当需要显示一个操作完成的百分比时。

API

属性 说明 类型 默认值
ma-size 尺寸 Int 120
ma-type 类型 line circle String line
ma-status 颜色状态 success danger warning String success
ma-percent 进度百分比 Int 0
ma-stroke-width 填充线条宽度 Int 6
