Dropdown 下拉菜单

当页面上的操作命令过多时，用此组件可以收纳操作元素。点击或移入触点，会出现一个下拉菜单。可在列表中进行选择，并执行相应的命令。

代码演示

API

属性 说明 类型 默认值
ng-model 必填 String -
ma-show 下拉菜单是否显示，双向绑定 boolean false
ma-text-key 下拉菜单每一项显示文本关键字 String text
ma-value-key 下拉菜单每一项对值关键字 String value
ma-show-hover 鼠标覆盖显示下拉 - -
ma-show-click 鼠标点击显示下拉 - -
ma-selected-hover 选中下拉后关闭下拉 - -
ma-items 下拉内容 JSON Array -
ma-item-click 下拉项点击事件 Function -
ma-null-text 没下拉数据时显示的内容，默认为暂无数据，不配不显示 String -
