API

属性 说明 类型 默认值
name 非必填 String -
ng-model 必填 String -
ma-placeholder 非必填 String -
ma-null-text 非必填，没有数据时的下拉提示文本 String 暂无数据
ma-search 非必填，是否显示下拉搜索 Boolean false
ma-clear 非必填，是否显示下拉清空 Boolean false
ma-multiple 非必填，是否为多选 Boolean false
ma-limit 非必填，多选数量限制 Int 无限制
ma-text-key 每一项显示文本关键字 String text
ma-value-key 每一项对应值关键字 String value
ma-data 选填，下拉所要显示的列表 Array -
ma-get-data 选填，返回下拉所要显示的列表，可以是 promise，也可以是Array， @params {$searchKey: String} Function -
