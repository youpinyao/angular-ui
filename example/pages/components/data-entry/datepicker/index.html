DatePicker 日期选择框

当用户需要输入一个日期，可以点击标准输入框，弹出日期面板进行选择。

jsfiddle

API

ma-date-picker

属性 说明 类型 默认值
ma-view 初始化显示的视图，year month date hours minutes String -
ma-min-view 最小显示的视图，year month date hours minutes String -
ma-format 显示时间格式化 moment 标准 String -
ma-min-date 最小可选时间 Date false
ma-max-date 最大可选时间 Date false
ma-clear 显示清空按钮 Boolean true

ma-date-range-picker

属性 说明 类型 默认值
ma-format 显示时间格式化 moment 标准 String -
ma-min-date 最小可选时间 Date false
ma-max-date 最大可选时间 Date false
ma-clear 显示清空按钮 Boolean true
ma-config 双向绑定 json http://longbill.github.io/jquery-date-range-picker/ JSON -
