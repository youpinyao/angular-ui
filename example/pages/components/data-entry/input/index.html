API

属性和 html input 一致，除了 disabled、readonly 由 ng-disabled、ng-readonly 代替。
属性 说明 类型 默认值
ma-clear 清空按钮 boolean -
ma-num 只要在标签内配置此属性就可以限制只能输入数字 - -
ma-decimal 和 ma-num 配合使用限制数字小数点后位数长度 Int -
ng-model 必填 String -
ma-icon-click 选填，在有右边搜索icon场景下使用 Function -
样式 说明
.ma-input-success 显示为正确
.ma-input-error 显示为错误
.ma-input-warning 显示为警告
.ma-input-search-normal 显示为搜索样式 正常
.ma-input-search-black 显示为搜索样式 黑色
