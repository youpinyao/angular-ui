Class 预设样式

提供一些预设的样式，方便编码过程

代码演示

属性 说明
.clearfix 清浮动
.ellipsis 单行超出隐藏
.center 文本居中
.fl 左浮动
.fr 右浮动
.mr-*, .ml-*, .mt-*, mb-*（*符号对应值为：-5, -10, -20, 0, 5, 10, 20） 预设上下左右 margin 属性
.pr-*, .pl-*, .pt-*, pb-*（*符号对应值为 0, 5, 10, 20） 预设上下左右 padding 属性
@mixin transition 范例：@include transition(all 300ms ease); 非必填，默认 all 300ms ease
@mixin transform 范例：@include transform(all 300ms ease); 必填
@mixin transform-origin 范例：@include transform(top center); 非必填，默认 top center
@mixin line-clamp 范例：@include lineClamp(2); 非必填，默认 2
@mixin ellipsis 范例：@include ellipsis();
@mixin unselect 范例：@include unselect(); 让作用域内不可选中
@mixin iconfont 范例：@include iconfont(); 设置为图标的字体
